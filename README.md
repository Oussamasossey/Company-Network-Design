# Secure Enterprise Network Infrastructure

## Project Overview

This project demonstrates the design and implementation of a secure enterprise network using Cisco Packet Tracer. The network was built following enterprise networking best practices, including segmentation, routing, security, and network services.

---

## Features

- VLAN Segmentation
- 802.1Q Trunking
- Router-on-a-Stick
- Inter-VLAN Routing
- DHCP Server
- DHCP Relay
- DNS Server
- SSH Remote Management
- Extended Access Control Lists (ACL)
- Port Security (Sticky MAC)
- NAT/PAT Configuration

---

## Network Topology

- VLAN 10 – ADMIN
- VLAN 20 – SALES
- VLAN 30 – IT
- VLAN 99 – MANAGEMENT
