# Project Scenario

A company requires a secure enterprise network with three departments:

- Administration
- Sales
- IT

The objectives are:

- Separate departments using VLANs
- Enable communication through Inter-VLAN Routing
- Automatically assign IP addresses using DHCP
- Provide DNS name resolution
- Secure remote management using SSH
- Restrict access to the Management VLAN using ACLs
- Protect access ports using Port Security
- Simulate Internet access using NAT/PAT
